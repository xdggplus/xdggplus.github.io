pyenv 安装

命令	说明
pyenv install --list	查看可安装的 Python 版本
pyenv install 3.9.7	安装指定版本的 Python
pyenv versions	查看已安装的 Python 版本
pyenv global 3.9.7	设置全局 Python 版本
pyenv virtualenv 3.9.7 myenv	创建虚拟环境
pyenv activate myenv	激活虚拟环境
pyenv deactivate	退出虚拟环境
pyenv uninstall myenv	删除虚拟环境
